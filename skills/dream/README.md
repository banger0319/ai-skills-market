# Dream Agent

强大的多模态 AI 智能体，支持复杂任务规划与执行。

## 功能特性

- 多模态理解：支持文本、图像、代码等多种输入
- 复杂任务规划：自动拆解复杂任务为可执行步骤
- 上下文感知：保持长上下文记忆，理解用户意图

## 使用场景

- 代码生成与重构
- 数据分析与可视化
- 文档编写与润色
- 多语言翻译
