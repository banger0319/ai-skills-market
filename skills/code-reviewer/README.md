# AI Code Reviewer

自动审查代码质量，提供改进建议和最佳实践提示。

## 功能特性

- 代码质量分析：检测潜在 bug、性能问题和安全漏洞
- 最佳实践建议：根据语言规范推荐改进方案
- 风格一致性检查：确保代码风格统一

## 使用场景

- Pull Request 自动审查
- 代码重构辅助
- 新成员代码质量培训
